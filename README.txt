# Simple Mystery Dungeon PWA v0.3.0

## v0.3.0 追加・修正
- 装備中アイテムを持ち物一覧でも緑色表示し、装備名の後に「:E」を表示
- 手持ち・倉庫の自動整頓を追加
- 整頓順: 装備中 → 武器 → 防具/盾 → 指輪 → 矢 → 回復アイテム → 食料 → 杖 → 巻物 → 壺
- Shift押下中は上下左右キー単体では移動せず、縦＋横の同時押しで斜め移動
- 指輪を右手・左手に1個ずつ装備可能
- 矢を実装
- 攻撃ボタンを追加
- 攻撃時に対象マスへエフェクトを表示
- 落ちている装備・敵ドロップ装備に+0〜+3を付与
  - +なし: 50%
  - +1: 25%
  - +2: 15%
  - +3: 10%
- 合成の壺を追加
- 武器/盾の特殊効果と、合成による特殊効果継承に対応
- 攻撃強化の巻物・防御強化の巻物を追加
- GAS URL組み込み済み
  - https://script.google.com/macros/s/AKfycbzUJb7b8I7w5HG7h7OeR-43vawtbcBiudTLO2qzOhOrt4O9IYxIRnhObWn9-n3Io5dUoA/exec

## ローカル確認
ZIPを展開して start_local_server_v0_3_0.bat を実行してください。
http://localhost:8000/mysterious_dungeon_pwa_v0_3_0/
